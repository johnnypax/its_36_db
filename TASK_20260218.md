# Task di connessione JDBC

Creare un sistema di inserimento e lettura di libri, ogni libro è caratterizzato da:
- Titolo
- Autore
- Descrizione
- Prezzo

PER INSERIMENTO: Il libro non dovrà essere creato sotto forma di oggetto semplice all'inizio, ma utilizzate lo Scanner per chiedere all'utente quali campi valorizzare.

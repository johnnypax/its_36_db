# Creare un sistema di gestione richieste di supporto
Voglio creare un sistema di gestione di supporto per il mio Tier 1, ogni richiesta di supporto ha:

- Codice identificativo
- Nominativo del cliente
- Titolo della richiesta
- Testo della richiesta di supporto
- Data di creazione richiesta (all'apertura)
- Status (Creata/In lavorazione/Risolta)
- Data di risoluzione
- Operazione svolta per la risoluzione

Voglio poter:
- Stampare il riepilogo delle richieste
- Stampare una singola richiesta per codice identificativo
- Aggiungere una richiesta
- Modificare una richiesta per codice identificativo (successivamente chiedere l'operazione svolta per la risoluzione ed assegnare in automatico la data).
